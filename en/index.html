Current tool is the latest version

Support for 64-bit and 32-bit Applications

Supports debugging of both 64-bit and 32-bit Windows applications, providing a unified debugging experience to meet different debugging needs.

Powerful Breakpoint System

Supports multiple breakpoint types including hardware breakpoints, software breakpoints, memory breakpoints, etc. Can set conditional breakpoints and log breakpoints for precise control of program execution flow.

Memory and Register Operations

Real-time viewing and modification of memory content and register status, supporting memory dumps and editing, helping to deeply analyze program runtime status.

Disassembly and Assembly Features

Provides a powerful disassembly engine, supports multiple instruction sets, can view and modify assembly code, and preview modification effects in real-time.

Plugin System

Supports plugin extension functionality, can add new debugging features and analysis tools through plugins to meet specific domain debugging needs.

Script Support

Built-in script engine, supports JavaScript for writing debugging scripts, can automate repetitive debugging tasks to improve debugging efficiency.

Open Source and Free

Completely open source and free, no hidden costs, community-driven development, continuously updated and improved, supporting Windows systems, ideal for reverse engineering and software analysis.
